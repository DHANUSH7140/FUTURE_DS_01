# Online Retail Sales Performance Dashboard

## Overview
This project was completed as part of my internship with Future Interns. It focuses on analyzing online retail transaction data to evaluate sales performance, identify revenue trends, and understand customer purchasing behavior using Power BI.

## Objective
- Monitor key sales KPIs
- Analyze revenue trends over time
- Understand customer activity
- Identify patterns to support business decision-making

## Key Features
- Interactive Power BI dashboard
- KPI tracking (Revenue, Orders, Customers, Quantity, Avg Order Value)
- Monthly sales trend analysis
- Data cleaning and transformation using Power Query
- DAX measures for business metrics

## Tools Used
- Power BI
- DAX
- Power Query
- Data Visualization

## Outcome
Demonstrates the ability to transform raw transactional data into actionable insights and build professional dashboards for performance monitoring.
